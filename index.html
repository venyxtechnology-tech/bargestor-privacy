Política de Privacidade

BarGestor  ·  Última atualização: 13 de maio de 2026

1. Introdução

A Venyx Technology é responsável pelo aplicativo BarGestor, um sistema de gestão desenvolvido para estabelecimentos do ramo de bares e restaurantes.

Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos as informações dos usuários. Ao utilizar o BarGestor, você concorda com os termos aqui descritos.

2. Informações que Coletamos

2.1 Dados de Autenticação

Para acessar o aplicativo, utilizamos o protocolo OAuth2 com PKCE. Durante a autenticação, processamos:

  • Nome de usuário e endereço de e-mail fornecidos no cadastro
  • Tokens de acesso temporários para manter a sessão autenticada
  • Identificador único do usuário no sistema

2.2 Dados Operacionais

Para o funcionamento do sistema de gestão, coletamos e armazenamos:

  • Pedidos realizados e itens consumidos
  • Informações de mesas, comandas e atendimentos
  • Dados de estoque e catálogo de produtos
  • Registros de operações e atividades no sistema

2.3 Dados de Dispositivo

Para fins técnicos e de suporte, podemos coletar informações sobre o dispositivo, como versão do sistema operacional Android e identificador do dispositivo para autenticação.

2.4 Bluetooth

O aplicativo solicita permissões de Bluetooth exclusivamente para comunicação com impressoras térmicas compatíveis para emissão de comprovantes e pedidos. Não coletamos, armazenamos nem transmitimos dados de outros dispositivos Bluetooth ao redor do usuário.

2.5 Localização

O aplicativo pode solicitar permissão de localização em versões Android 12 ou superior. Esta permissão é tecnicamente exigida pelo sistema operacional Android para a funcionalidade de varredura Bluetooth nessas versões. Não coletamos, usamos, armazenamos nem compartilhamos dados de localização para nenhuma outra finalidade.

3. Como Usamos as Informações

As informações coletadas são utilizadas exclusivamente para:

  • Autenticar e autorizar o acesso dos usuários ao sistema
  • Processar e registrar pedidos e operações do estabelecimento
  • Gerar relatórios e estatísticas operacionais para o gestor
  • Controle de estoque e catálogo de produtos
  • Comunicação com impressoras térmicas via Bluetooth
  • Garantir a segurança e o correto funcionamento do sistema

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros, exceto:

  • Quando exigido por lei, ordem judicial ou autoridade competente
  • Para proteger os direitos, propriedade ou segurança da Venyx Technology, dos usuários ou de terceiros

Os dados operacionais do estabelecimento são acessíveis apenas aos usuários autorizados pelo próprio estabelecimento contratante.

5. Armazenamento e Segurança

Adotamos medidas técnicas para proteger suas informações:

  • Comunicação criptografada via HTTPS/TLS
  • Autenticação com tokens JWT assinados digitalmente (RSA 2048 bits)
  • Tokens de sessão armazenados de forma segura no dispositivo
  • Acesso restrito aos dados por controle de permissões

Os dados são mantidos enquanto o contrato de uso estiver ativo e removidos conforme solicitação do estabelecimento após o encerramento.

6. Seus Direitos (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento de seus dados pessoais
  • Acessar os dados que possuímos sobre você
  • Solicitar a correção de dados incompletos ou inexatos
  • Solicitar a exclusão de dados desnecessários
  • Revogar o consentimento a qualquer momento

Para exercer qualquer um desses direitos, entre em contato pelo e-mail abaixo.

7. Retenção de Dados

Mantemos seus dados pelo tempo necessário para a prestação dos serviços e cumprimento de obrigações legais. Tokens de autenticação expiram automaticamente. Dados operacionais são mantidos conforme o contrato com o estabelecimento.

8. Menores de Idade

O BarGestor é destinado exclusivamente ao uso por profissionais adultos em estabelecimentos comerciais. Não coletamos intencionalmente dados de menores de 18 anos.

9. Alterações nesta Política

Podemos atualizar esta política periodicamente. Notificaremos os usuários sobre alterações significativas por meio do aplicativo ou por e-mail. O uso continuado após a publicação das alterações constitui aceitação da nova política.
